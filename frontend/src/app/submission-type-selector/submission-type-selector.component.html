Transcriptomes

Transcriptome assemblies contain transcript sequences and optional functional annotation.

info Unannotated sequences are submitted using a Fasta file.

info Annotated sequences are submitted using an EMBL-Bank flat file.

info Each transcriptome assembly is associated with a Study (Project) and a Sample. These must be submitted before the transcriptome assemblies are submitted.

info Transcriptome assemblies must be submitted with an info file. The info file is a text file with the following fields: 

        Assembly Name: PLEASE FILL
        Assembly Method: PLEASE FILL
        Sequencing Technology: PLEASE FILL

info The info and sequence files must be gzip compressed.
