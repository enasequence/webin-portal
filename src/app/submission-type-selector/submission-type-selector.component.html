Reads

Raw sequencing reads are submitted as runs. Each run is associated with read data files, experimental information, a study and a sample.

info Read data must be uploaded before runs can be submitted.

info Studies and samples must be submitted before runs are submitted.

Genomes

Genome assemblies can contain contigs, scaffolds, chromosomes, unlocalised sequences and optional functional annotation. Typically, each assembly is associated with a unique BioProject and BioSample combination. Assembly updates are detected automatically from this unique pairing.

info BioProjects and BioSamples must be submitted before genome assemblies are submitted.

Transcriptomes

Transcriptome assemblies contain transcripts and optional functional annotation. Transcriptome assemblies are submitted as a fasta file and a transcriptome assembly info file. If the transcripts are associated with functional sequence annotation then the sequences are submitted using a flat file.

Each transcriptome assembly is associated with a BioProject and a BioSample.

info BioProjects and BioSamples must be submitted before transcriptome assemblies are submitted.

info Transcriptome assemblies must be submitted with an info file. The info file is a text file with the following fields: 

        Assembly Name: PLEASE FILL
        Assembly Method: PLEASE FILL
        Sequencing Technology: PLEASE FILL
